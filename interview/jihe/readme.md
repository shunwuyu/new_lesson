- ES6 Set Map 区别是什么？
  ES6中的Set是数据结构集合
  Map 是数据结构Hash表

- 何为集合？
  集合是一种不允许值重复的顺序数据结构。
- 数据用什么存？
  对象  key 不重复
- 结合Set 语法 ADT 
  - 判断元素是否在集合中
    调用对象原型上的hasOwnProperty方法判断元素是否在对象中

  - 向集合中添加元素
    判断当前要添加的元素是否在集合中
    如果当前要插入的元素不在集合中则将要添加的元素当作key添加到集合中

  - 删除集合中的元素  
    判断当前要删除的元素是否在集合中
    如果在集合中，则删除当前集合中的元素

  - 清空集合
    集合指向空对象即可
  - 获取集合大小
    声明一个变量来存储集合大小，遍历集合，集合大小自增，结束遍历返回集合大小。
  - 集合中的所有元素
    声明一个数组用于存储集合中的每个元素
    遍历集合，将遍历到的元素放进数组中
